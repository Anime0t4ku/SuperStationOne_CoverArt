# SuperStation One – 9mm Half Height DVD Cover Template

This repository contains a **Photoshop (PSD) template** for creating **9mm half height DVD covers** for **SuperStation One** cases.

The template is intended as a **design base**. Add your own game artwork, titles, and text, then export the result as **JPEG or PNG** for printing.

---

## Printing dimensions

- **Final trim size:** 283 × 125 mm  
- **With 3 mm bleed:** 289 × 131 mm  
- **Recommended resolution:** 300 DPI  

---

## Case reference

The template dimensions are based on standard **9mm CD/DVD cases**.
